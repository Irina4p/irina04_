About Me

Nama saya Niken Triwulandari dari Surabaya, siswi SMA IPIEMS SURABAYA. Di sekolah, saya senang belajar hal baru melalui organisasi dan praktik untuk melatih komunikasi.

Meskipun di sekolah saya sedang fokus belajar IT seperti Excel, Website, Arduino, dan Coding untuk menambah ilmu, cita-cita saya tetap ingin terjun ke dunia kesehatan di masa depan untuk berkontribusi bagi masyarakat.

Di luar urusan sekolah, hobi saya adalah membaca buku, membuat vlog, memasak, mewarnai, menyanyi, dan dance. Selamat datang di portofolio sederhana saya!
